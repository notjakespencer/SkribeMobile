# Documentation

This directory contains technical documentation for the J2K project applications.

## Available Documentation

### Momentum Journaling App

- **[System Diagram](./momentum-system-diagram.md)** - Comprehensive system architecture diagram showing the components, data flow, and integration points for the Momentum daily journaling application.
- **[Use Case Diagram](./momentum-use-case-diagram.md)** - Use case diagram showing the key interactions between users and the system, including authentication, daily journaling, history viewing, and progress tracking features.

## About the Diagrams

The system diagrams use Mermaid notation, which renders beautifully on GitHub. You can view them directly in this repository, or use any Mermaid-compatible viewer.
